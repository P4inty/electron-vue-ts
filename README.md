# electron-vue-ts

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Compile ts-folder files
```
npm run tsc-app
```

### Compiles project and start electron
```
npm run start
```
